# Next.js TesloShop App

## To run locally
1. Install dependencies 
```
yarn
```
or
```
npm i
```

2. Duplicate .env.example on the root folder, rename it as .env and fill env vars as required

3. Run development database with docker
```
docker-compose up -d
```

4. Start local server
```
yarn dev
```
or
```
npm run dev
```
